# ADR 0034: Use Chinese-first documentation in v1

## Status
Accepted

## Context
The product requirements and clarification process are conducted in Chinese. The v1 documentation needs to preserve precise product semantics and operator guidance.

## Decision
AIOPROXY v1 documentation is Chinese-first. README, focused docs, and example configuration comments should use Chinese as the primary language.

## Consequences
- Documentation matches the user's working language.
- Product semantics are less likely to be distorted during translation.
- English documentation can be added later if needed.
